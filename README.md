# How's Dan?

A personal website built with Flask.

Live site: [hows-dan.onrender.com](https://hows-dan.onrender.com)

---

## Running Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/audaniel9/hows-dan.git
   cd hows-dan
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Start the development server:
   ```bash
   python app.py
   ```

4. Visit `http://127.0.0.1:5000` in your browser.

---

## Deployment

This app is hosted on [Render](https://render.com). Any push to the `master` branch will automatically trigger a new deployment.
